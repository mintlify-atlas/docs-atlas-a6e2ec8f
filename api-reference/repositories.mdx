## Endpoint

```
GET /api/github/repos
```

Returns a paginated list of repositories for the authenticated user, including both public and private repositories based on OAuth scopes.

## Authentication

Requires valid NextAuth session with GitHub OAuth access token.

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number to retrieve
</ParamField>

<ParamField query="per_page" type="integer" default="30">
  Number of items per page. Maximum value is 100.
</ParamField>

<ParamField query="sort" type="string" default="updated">
  Sort field for repositories.
  
  **Options:**
  - `created` - Sort by creation date
  - `updated` - Sort by last update date
  - `pushed` - Sort by last push date
  - `full_name` - Sort alphabetically by name
</ParamField>

<ParamField query="direction" type="string" default="desc">
  Sort direction.
  
  **Options:**
  - `asc` - Ascending order
  - `desc` - Descending order
</ParamField>

## Example Request

```bash cURL
curl -X GET "https://your-domain.com/api/github/repos?page=1&per_page=30&sort=updated&direction=desc" \
  -H "Cookie: next-auth.session-token=YOUR_SESSION_TOKEN"
```

```javascript JavaScript
const response = await fetch('/api/github/repos?page=1&per_page=30&sort=updated&direction=desc', {
  credentials: 'include'
});

const data = await response.json();
console.log(data.repositories);
```

```python Python
import requests

response = requests.get(
    'https://your-domain.com/api/github/repos',
    params={
        'page': 1,
        'per_page': 30,
        'sort': 'updated',
        'direction': 'desc'
    },
    cookies={'next-auth.session-token': 'YOUR_SESSION_TOKEN'}
)

data = response.json()
print(data['repositories'])
```

## Rate Limiting

This endpoint consumes GitHub API quota. Rate limit headers are included in the response:

```
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4999
X-RateLimit-Reset: 1234567890
```
