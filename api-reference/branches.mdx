## Endpoint

```
GET /api/github/branches
```

Returns a paginated list of branches for a specified GitHub repository.

## Authentication

Requires valid NextAuth session with GitHub OAuth access token.

## Query Parameters

<ParamField query="owner" type="string" required>
  Repository owner (username or organization name)
</ParamField>

<ParamField query="repo" type="string" required>
  Repository name
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number to retrieve
</ParamField>

<ParamField query="per_page" type="integer" default="30">
  Number of items per page. Maximum value is 100.
</ParamField>

## Response

<ResponseField name="branches" type="array">
  Array of branch objects
  
  <Expandable title="Branch object">
    <ResponseField name="name" type="string">
      Branch name (e.g., `main`, `develop`, `feature/new-ui`)
    </ResponseField>
    
    <ResponseField name="commit" type="object">
      Latest commit information
      
      <Expandable title="Commit details">
        <ResponseField name="sha" type="string">
          Commit SHA hash
        </ResponseField>
        
        <ResponseField name="url" type="string">
          GitHub API URL for the commit
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="protected" type="boolean">
      Whether the branch has protection rules enabled
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination metadata
  
  <Expandable title="Pagination details">
    <ResponseField name="page" type="number">
      Current page number
    </ResponseField>
    
    <ResponseField name="perPage" type="number">
      Items per page
    </ResponseField>
    
    <ResponseField name="hasNext" type="boolean">
      Whether there are more pages available
    </ResponseField>
    
    <ResponseField name="hasPrev" type="boolean">
      Whether previous pages exist
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash cURL
curl -X GET "https://your-domain.com/api/github/branches?owner=octocat&repo=hello-world&page=1&per_page=30" \
  -H "Cookie: next-auth.session-token=YOUR_SESSION_TOKEN"
```

```javascript JavaScript
const response = await fetch('/api/github/branches?owner=octocat&repo=hello-world&page=1&per_page=30', {
  credentials: 'include'
});

const data = await response.json();
console.log(data.branches);
```

```python Python
import requests

response = requests.get(
    'https://your-domain.com/api/github/branches',
    params={
        'owner': 'octocat',
        'repo': 'hello-world',
        'page': 1,
        'per_page': 30
    },
    cookies={'next-auth.session-token': 'YOUR_SESSION_TOKEN'}
)

data = response.json()
for branch in data['branches']:
    print(f"{branch['name']}: {branch['commit']['sha']}")
```

## Error Responses

### 400 Bad Request

Missing required parameters:

```json
{
  "error": "Missing required parameters: owner and repo"
}
```

### 401 Unauthorized

```json
{
  "error": "Unauthorized"
}
```

### 404 Not Found

Repository not found or no access:

```json
{
  "error": "Not Found"
}
```

### 500 Internal Server Error

```json
{
  "error": "Failed to fetch branches"
}
```

## Rate Limiting

This endpoint consumes GitHub API quota. Rate limit headers are included in the response:

```
X-RateLimit-Limit: 5000
X-RateLimit-Remaining: 4998
X-RateLimit-Reset: 1234567890
```
